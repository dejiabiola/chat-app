Project Information

This project is to showcase how I can use my knowledge of HTML, CSS and JavaScript to build a web pages. I also used Nodejs and SocketIO to build a chat application that allows multiple users send messages across to each other.

The home page and about pages are simple HTML and CSS templates to convey information in a Semantic manner to users of the application. I imported fonts from Google Fonts for the font family of texts in the website. I also used a mix of bootstrap styling and my own custom styling to style the pages.

The chat page is a lot more complex and utilizes a lot of JavaScript code to ensure it's functionality. Since, some boiler plate code which already handled the chat communication was provided, I focused on styling the chat page in a more presentable manner similar to how a standard chat application like Whatsapp would look like. I also extended certain JavaScript functionalities. These are:

  1. Using bootstrap toast notification components, I created a way for users on the chat page to be notified when someone joins or leaves the application. A toast notification displays this information for about 3 seconds before fading away.
  2. I added code to track users keyboard presses in order to figure out when they are typing a message in the input box then I displayed a text just above the input field so other users are notified that a certain user is typing.

Technologies Used

The technologies used for this project are detailed in the table below

S/N Technology Purpose
1 HTML HTML is used to define the meaning and strucure of web content. Semantic HTML was used across to website to ensure optimal acessibility and code readability.
2 CSS Cascading Style Sheets is used to describe the presentation of the website. It was used on this project to describe how elements should be rendered on the screen.
3 JavaScript JavaScript is an interpreted programming language that is to provide interactivity in web pages. It is also used in non browser environments such as Node.js to build server side applications. JavaScript is used extensively across this application especially for the chat page as it is the programming language in which the socket library is written in.
4 Bootstrap Bootstrap is a UI components and untilies library that helps developers rapidy build responsive mobile first websites. It is used in this applications to ensure mobile responsiveness, for the navigation bar across the HTML pages and also for margin and padding setting on different elements across the website.
5 Nodejs Nodejs is a JavaScript runtime built on Chrome's V8 JavaScript engine. It basically allows developers write and compile JavaScript code in a non browser environment, therefore allowing JavaScript to be used for building server side applications.
6 ExpressJs Express is a fast, unopinionated minimalist web framework for Nodejs. Due to the complexity of writing plain Nodejs code, express was leveraged to write the server code in an easier and faster way.
7 Socketio Socket.IO is a reliable, performant and scalable bidirectional communication linrary. It is used on this project to power the functionality of the chat application.
